# ainjiner.github.io

> Landing page dan pusat dokumentasi organisasi Ainjiner — AI Engineering Organization dari Indonesia.

**Live:** https://ainjiner.github.io

---

## Tentang Ainjiner

Ainjiner adalah organisasi open-source yang membangun tools AI/ML untuk developer dan pendidikan Indonesia. Semua proyek MIT licensed, self-hosted, dan community-driven.

---

## Proyek

| Proyek | Deskripsi | Status |
|--------|-----------|--------|
| [**AI Lab**](https://github.com/ainjiner/ai-lab) | ML/LLM Engineering Platform — provider registry, model catalog, experiment tracking, cost analytics | Active |
| [**2USE**](https://github.com/ainjiner/2USE) | Socratic AI middleware — menolak jawaban instan, mengajarkan berpikir kritis | Active |
| [**We Will Shine**](https://github.com/ainjiner/we-will-shine) | Platform bimbingan karier & wellbeing siswa Indonesia berbasis AI | Live MVP |

## Tech Stack (website ini)

- [Astro](https://astro.build) — static site generator
- [Tailwind CSS v4](https://tailwindcss.com) — styling
- [Bun](https://bun.sh) — runtime & package manager
- Deploy via [GitHub Actions](https://github.com/ainjiner/ainjiner.github.io/actions)

---

## Development

```bash
# Install dependencies
bun install

# Start dev server
bun dev

# Build for production
bun build

# Preview build
bun preview
```
